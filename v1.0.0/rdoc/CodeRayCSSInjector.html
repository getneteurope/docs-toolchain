    class CodeRayCSSInjector

    CodeRay CSS Injector

    By default, Asciidoctor will put the CodeRay CSS file in the body of the HTML as one of the very last tags. In addition, this will only be injected if the page has actual source code on it. Given, that we use the page switch functionality to only change the `div#content` when clicking on a new subsection, this tag needs to be on every page. Only then will every source code block show correctly, even if switching from a page without source code.

    This class will add the coderay-asciidoctor.css to every document head and remove the CSS link tag in the body, if it exists.

    Always add the link tag in the head after the others. Check for the link tag loading coderay-asciidoctor.css and remove it from the body.

def process(document, output)
  html = Nokogiri::HTML(output)
  coderay_css = 'css/coderay-asciidoctor.css'

  # get the coderay CSS if it exists and remove it
  coderay_css_tag = html.at_xpath('html/body/link')
  coderay_css_tag.remove if coderay_css_tag

  css_links = html.xpath('html/head/link')
  last_css_link = css_links.last
  # only add the link if it's not already present (index file gets processed twice)
  if (!last_css_link.nil?) && last_css_link.attr('href') != coderay_css
    css_links.last.add_next_sibling(
      '<link rel="stylesheet" href="css/coderay-asciidoctor.css">'
    )
  end

  return html.to_html
end
